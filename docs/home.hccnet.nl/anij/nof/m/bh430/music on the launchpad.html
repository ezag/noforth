Beethoven430

Beethoven430 is software for the Launchpad MSP430 with noForth C or V on it.
With the Beethoven430 program you can play "tunes".
  • Connect a loudspeaker (high impedance) to p2.4 and p2.5
  • Include (load) the beethoven430.f program
  • Download one of the tunes from this web site, or write your own tune
  • Include (load) the tune in noForth
  • Type the name of the tune
